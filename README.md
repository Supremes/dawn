
## Technology Stack

### Frontend:
- Core Framework: Vue 3 (Blog Frontend) / Vue 2 (Admin Backend)
- State Management: Pinia (Blog Frontend) / Vuex (Admin Backend)
- Routing: Vue Router
- HTTP Client: Axios
- Additional Technologies: See package.json in frontend projects

### Backend:
- Core Framework: Spring Boot
- ORM Framework: MyBatis Plus
- Security Framework: Spring Security
- Cache Middleware: Redis
- Message Queue: RabbitMQ
- Search Engine: Elasticsearch
- Object Storage: MinIO
- Job Scheduler: Quartz
- Additional Technologies: See pom.xml in backend project
